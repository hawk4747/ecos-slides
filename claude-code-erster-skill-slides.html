Wie Claude Skills lädt

01
Liste sehen
Nur Name und Beschreibung von jedem Skill — immer sichtbar.
02
Match auf Prompt
Dein Prompt passt zur Beschreibung eines Skills.
03
Skill nachladen
Erst jetzt landet der volle Skill im Kontext.

3 Wege einen Skill zu triggern

OPTION 1
Natural Language
"Schreib mir eine Produktbeschreibung." Hoffen dass Claude den richtigen Skill findet.
OPTION 2
Expliziter Hinweis
"Nutz den Meta-Ads Skill für den Text." Zwingt Claude in die Richtung.
OPTION 3
Slash-Command
/meta-ads — Claude lädt den Skill garantiert.

Wann lohnt sich ein eigener Skill

01
Wiederholung
Du machst die Aufgabe mindestens jede Woche.
02
Eigener Stil
Claude würde es ohne Info anders machen als du willst.
03
Eigenes Wissen
Brand, Kunden, Frameworks — Kontext aus deiner Welt.

Anatomie eines Skills

meta-ads-copy/
├── SKILL.md // pflicht
├── references/ // optional, für Details
└── scripts/ // optional, für Code

User Scope vs Project Scope

USER SCOPE
In allen Projekten
~/.claude/skills/
Liegt in deinem Home-Ordner.
Skill ist überall verfügbar.
PROJECT SCOPE
Nur in diesem Projekt
mein-shop/.claude/skills/
Liegt im Projekt-Ordner.
Skill nur dort verfügbar.
